# parel.dev

## Quick Start

```bash
# Install & run
npm install
npm run dev

# Build
npm run build
```

## Project Structure

```
src/
  ├── components/      # React components
  ├── lib.ts          # Data, schemas, utilities
  ├── styles/         # CSS with custom properties
  ├── App.tsx         # Main layout
  └── main.tsx        # Entry point

config/               # ESLint, Prettier, TypeScript
public/              # Static files, PWA
```
